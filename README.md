# SeivRA E-com demo

A premium Django + PostgreSQL ecommerce demo project being built phase by phase with a portfolio-grade engineering approach.

## Planned Core Features

- Homepage
- Category listing
- Product listing
- Product detail page
- Search and filter
- Cart and wishlist
- Checkout and mock payment
- Signup, login, logout
- Customer profile and order history
- Reviews and ratings
- Coupon support
- Blog section
- Professional admin management
- Analytics and reporting direction

## Planned Stack

- Django
- PostgreSQL
- Django Templates
- Git and GitHub
- VS Code
- Future-ready DRF expansion path
